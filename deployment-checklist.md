## 🛠️ 技術的詳細情報

### 現在のアプリ構成
- **フロントエンド**: React Native + Expo Router
- **状態管理**: AsyncStorage (オフライン対応)
- **UI**: 食べログ風デザインシステム (ピンク #ea5a7b)
- **機能**: 6つの主要機能 (認証, 地図, 検索, レビュー, 通知, お気に入り)

### ビルド済みファイル
- **場所**: `dist/` フォルダ
- **サイズ**: 約1.59MB (メインバンドル)
- **アセット**: フォント、アイコン含む37ファイル

### 現在の制限事項
- モックデータ使用（実際のAPIは未接続）
- テストカバレッジなし
- PWA機能無効
